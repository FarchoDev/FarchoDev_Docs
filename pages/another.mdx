
## **Fundamentos de PHP y Programación Orientada a Objetos (POO)**

### Instalación y Configuración de Entorno de Desarrollo
- **Instalar PHP y un servidor local**: XAMPP, WAMP, MAMP o Laravel Homestead.
- **Familiarizarse con Composer**: Instalación y configuración.
- **Primeros pasos con PHP**: Crear un archivo `.php`, ejecutar en el navegador, y probar comandos básicos.

### Fundamentos de PHP
- **Variables y tipos de datos**: Enteros, cadenas, arrays, booleans, etc.
- **Operadores y expresiones**: Aritméticos, comparación, lógicos y asignación.
- **Estructuras de control**: `if`, `else`, `elseif`, `switch`, `for`, `while`, `foreach`.
  
### Programación Orientada a Objetos en PHP
- **Clases y objetos**: Crear clases, instanciar objetos.
- **Propiedades y métodos**: Definir métodos y propiedades, visibilidad (`public`, `private`, `protected`).
- **Herencia y polimorfismo**: Crear clases que hereden y sobreescriban métodos.
- **Interfaces y traits**: Aplicar interfaces y traits en una clase.

## **Autenticación y Autorización**

### Implementación de Autenticación

- **Crear una interfaz de usuario básica** para login y registro.

### Middleware de Autenticación
- **Aplicar Middleware**: Restringir acceso a ciertas rutas usando `auth` y `guest`.
- **Redirección de usuarios**: Configurar la redirección después del login y logout.

### Autorización con Roles y Permisos
- **Roles y Permisos básicos**: Crear roles (ej. admin, usuario) y middleware para limitar acceso.
- **Uso de políticas de autorización**: Crear y usar políticas para verificar permisos a nivel de modelo.
