## [Módulo 3: Uso básico de Git](/git/modulo-3)

### [3.1. Iniciar y clonar repositorios](/git/modulo-3)

- #### [`git init`: Crear un Nuevo Repositorio Git](/git/modulo-3)
- #### [`git clone`: Clonar un Repositorio Git](/git/modulo-3)

### 3.2. Gestión básica de cambios y archivos:

- #### `git add`: Añadir Archivos al Área de Preparación**
- #### `git commit`: Guarda los cambios en el repositorio
- #### `git status`:Verificar el Estado del Repositorio
- #### `git diff`: Verificar las Diferencias entre Cambios
- #### `git reset`: Deshacer Cambios y Modificar el Historial
- #### `mkdir`: Crear Directorios
- #### `touch`: Crear y Modificar Tiempos de Archivos
- #### `mv`: Mover y Renombrar Archivos o Directorios
- #### `rm`: Elimina archivos o directorios.

### 3.3. Exploración del historial y navegación:

- #### `git log`: Ver el Historial de Commits
- #### `git checkout`: Cambiar de Rama o Restaurar Archivos
- #### `git branch`: Gestionar Ramas en Git
- #### `git merge`: Fusionar Ramas en Git
- #### `git rebase`: Reorganizar el Historial de Commits
- #### `cd`: Cambiar el Directorio de Trabajo
- #### `ls`: Listar Contenidos de un Directorio
- #### `pwd`: Imprimir el Directorio de Trabajo Actual

### 3.4. Trabajo con repositorios remotos:

- #### `git remote add`: Añadir un Repositorio Remoto
- #### `git remote -v`: Verificar Repositorios Remotos
- #### `git push`: Enviar Cambios al Repositorio Remoto
- #### `git pull`: Obtener y Fusionar Cambios desde el Repositorio Remotoz
- #### `git fetch`: Obtiene cambios desde un repositorio remoto, pero no los fusiona automáticamente.
- #### `git remote rm`: Elimina un repositorio remoto de la configuración.

### 3.5. Funcionalidades avanzadas:

- #### `git cherry-pick`: Aplica un commit específico a la rama actual.
- #### `git revert`: Deshace un commit específico creando un nuevo commit de reversión.
- #### `git stash`: Guarda cambios locales temporalmente para trabajar en otra cosa.
- #### `git tag`: Crea, lista o elimina etiquetas (versiones específicas) en el repositorio.

### 3.3. Historial y navegación. 

- #### `git log`.
- #### `git checkout`.
